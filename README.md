# Tanishka Soni - Portfolio Website

A premium, high-performance portfolio website built with the MERN stack developer in mind.

## 🚀 Features
- **Modern UI**: Dark mode, glassmorphism, and neon accents.
- **Responsive**: Fully optimized for mobile, tablet, and desktop.
- **Interactive**: Smooth scrolling, typing effects, and hover animations.

## 🛠️ Tech Stack
- HTML5
- CSS3 (Variables, Flexbox, Grid)
- Vanilla JavaScript (ES6+)
- Font Awesome & Google Fonts

## 📦 How to Run
Simply open `index.html` in your web browser.

Or run with a local server:
```bash
# Python
python -m http.server 8000

# Node (http-server)
npx http-server .
```
